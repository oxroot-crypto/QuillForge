功能特性

将现代编辑器体验与 AI 辅助写作深度融合，为网文作者量身打造

📚

书籍与章节管理

层级化组织：书籍 → 章节，每本书独立配置世界观、剧情总结、角色档案，自动作为 AI 上下文。

✍️

AI 幽灵文本续写

AI 续写内容以幽灵文本形式出现在编辑器光标处。Tab 接受，Esc 取消——类似 IDE 代码补全。

🔍

AI 审阅 & 一致性检查

选中文本一键审阅，从文法、节奏、角色一致性、情节逻辑等维度给出具体建议，附带大纲遵循度评估。

💡

灵感构思

描述当前情节节点或卡点，AI 结合已有世界观和角色给出 3–5 个创意发展方向。

⚡

AI 生成章节

从零生成完整章节，支持篇幅选择、重新生成并覆盖应用到同一章节，自动保存快照。

🔄

AI 改写

选中文本或全章改写，支持自定义改写要求。自动保存快照，支持不同风格重新生成。

📋

大纲管理

每章独占大纲，标题从章节自动同步，描述自由编辑。支持拖拽排序、AI 一键生成大纲内容。

📸

版本快照

手动与自动快照，每条快照保存标题+内容。支持预览、恢复、单条删除，版本对比回溯无忧。

🎯

写作目标

设定每日字数目标，实时进度条显示完成度，追踪连续写作天数（🔥 streak）和写作时长。

🏷️

章节状态

彩色状态标识（草稿·修改中·已完成·冻结），点击切换，按状态筛选，一目了然。

🔌

多 AI 提供商

OpenAI · Anthropic · Ollama · 兼容接口（DeepSeek、Qwen 等）。保存为预设，工具栏快速切换。

📊

写作分析

句长分布、对话比例、可读性评分、词频统计，每本书独立数据分析。

🛡️ 安全

你的数据完全由你掌控

API Key 使用 AES-256-GCM 加密，存储在 Tauri 安全存储中
密钥不出 Rust 后端——前端仅知悉是否已配置
所有 LLM HTTP 请求均由 Rust 层代理，Key 不暴露给前端网络层
所有数据本地存储，无需注册账号，无需云服务

🚀 快速开始

从源码构建 QuillForge

# 安装依赖
$ npm install

# 启动开发模式
$ npm run tauri dev

# 生产构建
$ npm run tauri build
前置条件：Node.js ≥ 18 · Rust ≥ 1.70 · 各平台 Tauri 依赖
